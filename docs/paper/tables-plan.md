## 표 작성 원칙

- 실제 계산 전에는 `후보`로만 기록한다.
- synthetic 결과와 공개/샘플 DEM·DSM 결과를 분리한다.
- 실제 링크상태 검증값이 없으면 통신품질 성능표로 쓰지 않는다.
