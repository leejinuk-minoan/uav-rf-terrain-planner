# PR Reviews

This directory stores individual PR review records created after Task 014.

Use `docs/paper/templates/pr-review-template.md` for new entries. Keep each file focused on one PR, branch, head/base commits, changed files, acceptance criteria, boundary check, CI result, issues found, fix requests, final decision, merge commit, public repository sensitivity check, and follow-up tasks.

Legacy accumulated PR review records remain in `docs/paper/pr-review-log.md` as historical archive and legacy index material.
