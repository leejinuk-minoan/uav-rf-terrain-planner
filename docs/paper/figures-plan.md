- 실제 생성되지 않은 그림은 `후보`로만 기록한다.

- 실제 드론 운용 결과처럼 보이는 표현을 쓰지 않는다.
- 지도 화면 캡처는 Claude Code의 로컬 UI 확인 이후 사용한다.
