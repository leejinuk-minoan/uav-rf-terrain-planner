# Limitations

책임 주체: GPT Master Agent  
목적: 연구·구현·검증의 한계를 논문 작성 단계에서 과장 없이 관리한다.

---

## 1. 핵심 한계

| 구분 | 한계 | 논문 표현 기준 |
|---|---|---|
| 실제 드론 운용 | 수행하지 않음 | 연구·교육·시뮬레이션용 방법론 |
| 실제 링크상태 | RSSI/SINR/packet loss 미측정 | 통신품질 보장 주장 금지 |
| 점수식 가중치 | 실제 데이터로 최적화되지 않음 | 초기 휴리스틱 가중치 |
| DEM/DSM 품질 | 자료 해상도와 최신성에 영향 | 데이터 품질 의존성 명시 |
| DSM 중복 반영 | LOS/Fresnel과 중복 가능 | DEM-only/DSM-integrated/Hybrid 비교 필요 |
| 주파수 모델 | 단순 Fresnel 중심 | 송신출력, 안테나, 수신감도 등 미반영 |
| 경로 추천 | 경로 후보 시뮬레이션 | 실제 비행 가능성 보장 아님 |
| UI 지도 | 시각화 도구 | 작전 지시 도구 아님 |

## 4. 향후 연구

1. 실제 DEM/DSM 기반 검증 확대
2. 공개 또는 승인된 지형자료 기반 사례지역 분석
3. 가중치 민감도 분석
4. 점수항 ablation 분석
5. 실제 링크상태 데이터 확보 시 RSSI/SINR/packet loss와 비교
6. radio map 또는 ray-tracing 기반 보정
7. UI 사용성 평가
8. 계산 성능 및 격자 해상도 최적화
