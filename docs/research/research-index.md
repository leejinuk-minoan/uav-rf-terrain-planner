## 1. 프로젝트와 선행연구의 연결 구조

본 프로젝트는 사용자가 정찰목표지점 MGRS, 드론 운용반경, 허가 운용고도 AGL, 통신 주파수 대역을 입력하면 지형·전파 차폐 위험을 계산하여 발진기지 후보와 경로 후보를 제시하는 프로그램이다.

선행연구는 다음 세 축으로 반영한다.

| 축 | 선행연구 분야 | 프로그램 반영 |
|---|---|---|
| 전파 물리 | LOS, Fresnel Zone, 회절 모델 | 차폐위험 점수의 물리적 근거 |
| 지형 전파모델 | Longley-Rice / ITM | 고도화 단계의 전파감쇠 예측 모델 |
| UAV 경로계획 | Radio Map, Connectivity-Aware Planning | 차폐위험지도 기반 경로탐색 |

## 2. 핵심 선행연구 및 표준

### 2.1 LOS 기반 지형 차폐 분석

LOS(Line of Sight)는 송신점과 수신점 사이에 직접 가시선이 확보되는지를 판단하는 기본 개념이다. 드론 운용에서 발진기지 또는 조종자 위치와 드론 또는 목표 방향 사이에 능선, 산지, 건물, 수목 등이 위치하면 직접 가시선이 차단될 수 있다.

프로그램 반영 방식:

1. 발진기지 후보점과 정찰목표점 또는 경로점 사이의 직선 단면을 생성한다.
2. 단면상의 DEM/DSM 높이를 샘플링한다.
3. 통신선 높이와 중간 장애물 높이를 비교한다.
4. 장애물이 통신선을 넘으면 직접 차폐로 판단한다.

```text
지형 또는 장애물 높이 > 통신선 높이 → LOS 차단
지형 또는 장애물 높이 ≤ 통신선 높이 → LOS 확보
```

### 2.2 Fresnel Zone

무선통신은 수학적 직선 하나만을 따라 전달되는 것이 아니라, 송수신점 사이 직선 주변의 일정 공간을 함께 사용한다. 이 공간을 Fresnel Zone으로 볼 수 있다. LOS가 직접 차단되지 않더라도 제1 Fresnel Zone에 지형이나 장애물이 침범하면 신호 품질 저하가 발생할 수 있다.

프로그램 반영 방식:

| 판단 상태 | 의미 | 프로그램 위험도 |
|---|---|---|
| LOS 직접 차단 | 장애물이 통신선을 초과 | 높음 |
| LOS 확보, Fresnel 침범 | 직접선은 열려 있으나 전파 공간이 부족 | 중간 |
| LOS 및 Fresnel 여유 확보 | 통신선과 주변 공간 모두 확보 | 낮음 |

사용자가 입력한 통신 주파수 대역은 Fresnel Zone 반경 계산에 사용한다. 주파수가 낮을수록 Fresnel Zone 반경이 커지므로 동일 지형에서도 차폐위험이 더 크게 나올 수 있다.

### 2.3 ITU-R P.526 계열

ITU-R P.526 계열은 장애물에 의한 전파 회절과 Fresnel Zone 여유를 다루는 전파 모델 계열이다. 본 프로젝트에서는 전체 권고안을 직접 구현하는 것이 아니라 다음 개념을 차폐점수 설계에 반영한다.

1. 직접 가시선 차단 여부
2. 장애물의 상대 높이
3. Fresnel Zone 침범률
4. 침범 지점의 거리상 위치
5. 침범 구간의 길이 또는 비율

1차 MVP에서는 P.526의 모든 회절 손실식을 구현하지 않고, LOS/Fresnel 기반 정량 점수로 단순화한다. 이후 검증 데이터가 축적되면 회절 손실 모델을 추가한다.

### 2.4 Longley-Rice / ITM

Longley-Rice 또는 ITM(Irregular Terrain Model)은 지형 프로파일을 이용하여 전파 감쇠를 추정하는 대표적 모델이다. 지형고도, 주파수, 송수신점 높이, 거리 등을 반영할 수 있으므로 산악·구릉 지형에서 전파경로 평가에 활용할 수 있다.

프로그램 반영 계획:

| 단계 | 반영 수준 |
|---|---|
| 1차 MVP | LOS/Fresnel 기반 차폐위험 점수 |
| 2차 | 지형 단면 기반 감쇠 추정식 추가 |
| 3차 | ITM 라이브러리 또는 외부 모델 연동 |
| 4차 | 실제 링크상태 피드백 기반 보정 |

ITM은 개발 초기부터 필수 구현 대상으로 두지 않는다. 이유는 모델 파라미터가 많고 검증이 필요하기 때문이다. 초기에는 사용자가 이해하고 검증하기 쉬운 LOS/Fresnel 중심 모델이 적합하다.

### 2.5 Radio Map Based UAV Path Planning

Radio map 기반 UAV 경로계획 연구는 공간을 2D 또는 3D 격자로 나누고 각 격자에 통신품질 값을 부여한 뒤, 통신품질 제약을 만족하는 경로를 그래프 탐색 문제로 변환한다.

본 프로젝트의 반영 방식:

```text
DEM/DSM 기반 차폐위험지도
→ 임시 radio map 역할
→ 격자별 비용 부여
→ A* 또는 Dijkstra 경로탐색
→ 차폐 위험이 낮은 경로 후보 산출
```

초기에는 실제 RSSI, SINR, packet loss 측정값이 없으므로 terrain-derived shielding score를 radio quality의 대체 지표로 사용한다. 이후 실제 운용 피드백이 축적되면 경험적 radio map으로 확장한다.

### 2.6 Connectivity-Aware UAV Path Planning

Connectivity-aware path planning 연구는 비행경로의 거리뿐 아니라 연결성 유지 여부를 제약조건 또는 비용함수로 반영한다. 본 프로젝트에서는 차폐위험이 높은 영역을 통신 연결성이 낮은 영역 또는 coverage hole로 취급한다.

반영 방식:

1. 격자별 차폐위험 점수를 생성한다.
2. 고위험 격자에 높은 비용을 부여한다.
3. 운용반경을 초과하는 격자는 통과 불가 처리한다.
4. 차폐위험이 낮은 경로를 우선 탐색한다.
5. 사용자가 비교할 수 있도록 경로 후보 3개를 제시한다.

| 입력값 | 선행연구 반영 이유 |

## 4. 알고리즘 반영안

### 4.1 발진기지 차폐점수

```text
차폐안정성 점수 = LOS 점수 × 0.50 + Fresnel 여유 점수 × 0.35 + DSM 장애물 점수 × 0.15
```

#### LOS 점수

- 통신선 직접 차단 없음: 높은 점수
- 일부 구간에서 통신선 접촉 또는 근접: 중간 점수
- 명확한 직접 차단: 낮은 점수

#### Fresnel 여유 점수

- 제1 Fresnel Zone 여유 충분: 높은 점수
- 일부 침범: 중간 점수
- 대규모 침범: 낮은 점수

#### DSM 장애물 점수

- 건물/수목 표면 침범 없음: 높은 점수
- 부분 침범: 중간 점수
- 지속적 침범: 낮은 점수

### 4.2 발진기지 종합점수

```text
발진기지 종합점수 = 차폐안정성 점수 × 0.80 + 거리점수 × 0.20
```

거리보다 통신 안정성이 핵심이므로 차폐안정성 점수를 우선한다.

### 4.3 경로비용

```text
경로비용 = 차폐위험 비용 × W_shield + 실 비행거리 비용 × W_dist + 위험구간 페널티
```

| 후보 | W_shield | W_dist | 목적 |
|---|---:|---:|---|
| 1안 | 0.90 | 0.10 | 차폐 최소 |
| 2안 | 0.70 | 0.30 | 거리-차폐 균형 |
| 3안 | 0.85 | 0.15 | 우회 안정 |

## 5. 검증해야 할 연구 질문

1. LOS/Fresnel 기반 차폐점수가 실제 링크 안정성과 상관관계를 가지는가?
2. 주파수 대역별 Fresnel Zone 침범률이 추천 결과를 얼마나 바꾸는가?
3. DEM만 사용한 결과와 DSM까지 사용한 결과는 어느 정도 차이가 나는가?
4. 고도 AGL 값 변화가 발진기지 추천 순위에 어떤 영향을 주는가?
5. 차폐 최소 경로가 실제 비행거리 증가를 어느 정도 유발하는가?
6. 500m 단위 경유점 표기가 사용자의 경로 이해와 검토에 충분한가?

---

## 6. 향후 확장 방향

1. 실제 운용 또는 시뮬레이션 피드백을 수집한다.
2. 각 경로점의 실제 통신상태를 기록한다.
3. 차폐점수와 실제 링크상태의 차이를 분석한다.
4. 격자별 보정값을 누적하여 경험적 radio map을 만든다.
5. ITM 또는 더 정교한 전파모델을 선택적으로 연결한다.
6. 경로추천 모델의 가중치를 자동 보정한다.

## 7. 요약

본 프로젝트의 선행연구 반영 방향은 다음과 같다.

```text
LOS/Fresnel/회절 이론
→ 지형·주파수 차폐점수 설계

Longley-Rice / ITM
→ 고도화 단계의 지형 전파감쇠 예측

Radio Map 기반 UAV 경로계획
→ 차폐위험지도 기반 그래프 탐색

Connectivity-Aware Path Planning
→ 통신 안정성을 유지하는 경로 후보 생성
```

따라서 1차 MVP는 복잡한 전파모델보다 검증 가능한 LOS/Fresnel 기반 점수모델로 시작하고, 이후 실제 피드백과 ITM 기반 모델을 단계적으로 결합하는 것이 적절하다.
